## Квартира

- 2-комнатная, Святой Влас, Болгария
- Панорамный вид на море
- Балкон (зона отдыха — в разработке)
- Лифт в доме
- Паркинг 1 место под домом
- Полное оснащение (бельё, посуда, стиралка, печь, микроволновка)

**Правила:** Без курения / Без животных / Дети от 7 лет
